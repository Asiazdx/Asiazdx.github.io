二、算数运算符

javascript提供9个算数运算符，用来完成基本的算术运算。

  • 加法运算符（x+y）
  • 加法运算符（x-y）
  • 乘法运算符（x*y）
  • 除法运算符（x/y）
  • 余数运算符（x%y）
  • 自增运算符（++x或者x++）
  • 自减运算符（–x或者x–）
  • 数值运算符（+x）
  • 负数运算符（-x）

减法、乘法、除法运算符比较单纯，就是执行相应的数学运算。下面介绍其他几个算术运算符。

2、自增和自减运算符

自增和自减运算符，是一元运算符，只需要一个运算子。它们的作用是将运算子首先转为数值，然后加上1或者减去1.它们修改原始变量。

自增和自减运算符有一个需要注意的地方，就是放在变量之后，会先返回变量操作前的值，在进行自增/自减操作；放在变量之前，会先进行自增/自减操作，再返回变量操作后的值。

六、位运算符

位运算符用于直接对二进制进行计算，一共有7个。

  • 或运算（or）：符号为（|），表示两个二进制位都为0，则结果为0，否则为1。
  • 与运算符（and）：符号为&,表示若两个 二进制位都为1，则结果为1，否则为0。
  • 否运算（not）：符号为~，表示对一个二进制位取反位
  • 异或运算（xnot）：符号为^,表示若两个二进制位不相同，则结果为1，否则为0。
  • 左移运算（left shift）：符号为<<，详见下文解释
  • 右移运算（right shift）：符号为>>,详见下文解释
  • 带符号位的右移运算：符号为>>>,详见下文解释

这些位运算直接处理每个比特位（bit），所以是非常底层的运算，好处是速度极快，缺点是很不直观，许多场合不能使用它们，否则会使代码难以理解和差错。

有点需要特别注意，位运算只对整数起作用，如果一个运算不是整数，会自动转为整数后再执行。另外，虽然在javascript内部，数值都是以64位浮点数的形式存储，但是做位运算的时候，是以32位符号的整数进行运算的，并且返回值也是一个32为符号的整数。

2. “或运算”与“与运算”

这两种比较容易理解，就是逐位比较两个运算子。“或运算”的规则是，两个二进制之中有一个为1，就返回1，否则返回0。“与运算”的规则是，两个二进制位之中只要有一个位为0，就返回0，否则返回1。

上面两个表达式，0和3的二进制形式分别是00和11，所以进行“或运算”会得到11（即3），进行“与运算”会得到00（即0）。
位运算只对整数有效，遇到小数时，会将小数部分舍去，只保留整数部分。所以将一个小数与0进行或运算，等同于对该去除小数部分，即整数位。

需要注意的是，这种取整方法不适用超过32位整数最大值2147483647的数。
