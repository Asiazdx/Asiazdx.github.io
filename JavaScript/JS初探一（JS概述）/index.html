JS初探一（JS概述）

发表于 2016-08-01 | 分类于 JavaScript


如何在静态网页中做出如下动态效果：

  1. 亲切的欢迎讯息
  2. 漂亮的数字钟
  3. 有广告效果的跑马灯及简易的选举
  4. 轮流播放图片
  5. 对客户端数据进行验证
    等等……
    这些效果都是需要通过JavaScript来完成，当然JavaScript的用处不仅仅是这些。接下来我们就对JavaScript进行完整详细的介绍。

    一、简介

    1.什么是JavaScript

    JavaScript主要用于HTML页面，嵌入在HTML源码中。JavaScript是因特网上最流行的脚本语言，它存在于全世界所有的Web浏览器中，能够增强用户与Web站点和Web应用程序之间的交互。
    JS是弱类型语言，没有类型声明，它的变量不必具有一个明确的类型。JS是脚本语言。换句话说，能够用来编程的并且直接执行源代码的语言，就是脚本语言。
    JS是一种基于对象（Object）和事件驱动（Event Driven）并具有安全性能的脚本语言，可以广泛用于服务器、PC、笔记本电脑、平板电脑和只能手机等设备。
    HTML5的出现更是突出了JavaScript的重要性，例如HTML5的绘图支持，本地存储，高级应用，客户端通讯等，都大量使用了JavaScript。

    2、JavaScript的历史

    网景公司在上个世纪的1995年，凭借Nvaigator浏览器，成为Web时代开启最著名的第一代互联网公司。由于网景公司希望在静态的HTML页面添加一些动态的效果，于是叫Brendan Eich这个程序员在两周之内设计出了JavaScript语言。为什么起名叫javascript？原因是当时Java语言非常红火，所以网景公司希望能用java的名气来推广。但事实上，javascript除了语法上有点像Java，其他部分基本上没啥关系。

3、JavaScript的特点

  1. 简单性：它是基于Java基本语句和控制流之上的简单而紧凑的设计，是学习java的好过渡，而且，它的变量类型是采用弱类型，未采用严格的数据类型。
  2. 安全性：JS不允许访问本地硬盘，不能将数据存入到服务器上，不允许对网络文档进行修改和删除，只能通过浏览器实现信息浏览和动态交互，从而有效的防止数据的丢失。
  3. 动态性：JS可以直接对用户或客户输入做出响应，无效经过Web程序。它对用户的响应通过以事件驱动的方式进行，即由某种操作响应的事件响应，如：点击鼠标，移动窗口，选择菜单等。
  4. 跨平台：JS依赖浏览器本身，与操作环境无关。只要能运行浏览的计算机，并安装了支持JS的浏览器就可以正确执行，从而实现了“编写一次，走遍天下的梦想”。

4、JavaScript的组成

JavaScript是由ECMAScript，文档对象对象模型（DOM）、浏览器对象模型（BOM）三部分组成，其中核心是ECMAScript，它描述了该语法和基本对象；DOM描述了处理网页的内容的方法和接口，通过DOM，可以访问所有HTML元素，连同它们所包含的文本和属性，可以对其中的内容进行修改和删除，同时也可以创建新的元素；BOM描述了与浏览器的进行交互的方法和接口，BOM提供了独立于内容而与浏览器窗口进行交互的对象，例如可以移动，调整浏览器窗口的大小的window对象，可以用于导航的location对象和history对象，可以获取浏览器，操作系统与用户屏幕信息的nevigator与screen对象，可以使用document作为访问HTML文档的入口，管理框架的frames对象等。

3、JS输出方式

  1. 页面输出：document.write(‘hello world’)
    document可以在页面输出标签的形式。
  2. 控制台输出：console.log(‘hello worlf’)
    只能将标签一字符串的形式输出
  3. 提示框输出：alert(‘hello world’)
    只能将标签以字符串的形式输出

四、JS标示符

含义：一串字符串（字符串未必是标识符）
作用：用来给变量函数等命名的

标识符的规则

  1. 由字母，数字，下划线和$组成
  2. 不能以数字开头
  3. 不能是JS中的关键字和保留字
  4. 区分大小写
  5. 遵循驼峰原则，第一个首字母小写，后面单词首字母大写
  6. 见名知意

五、补充

字符集

javascript程序是用Unicode字符集编写的，Unicode是ASCII和Latin-1超集，并支持地球上几乎所有在用的语言。扩展阅读

区分大小写

javascript是区分大小写的。也就是说，关键字，变量，函数名和所有的标识符都必须采取一致的大小写形式。
但是需要注意的是，HTML不区分大小写。由于它和客户端javascript联系紧密，因此这点区别容易混淆。许多客户端javascript对象和属性与他们所表示的HTML标签属性名相同。在HTML中，这些标签和属性名可以使用大写也可以使用小写，而在javascript中则必须是小写，例如，在HTML中设置事件处理程序时，onclick属性可以写成onClick，但是在javascript代码中，必须使用小写的onclick。那么CSS是否区分大小写呢？经测验得到如下结论：

  • id名和class名是区分大小写的
  • 选择器中，只有标签选择器不区分大小写
  • 属性名和属性值是不区分大小写的

字面量

所谓的字面量（也可以称直接量），就是程序中直接使用的数据值。字面量只代表自身，不存储在特定位置。javascript中的字面量有：字符串，数字，布尔值，对象，数组，函数，正则表达式，以及特殊的null值。

undefined不是字面量：扩展阅读

严格模式

在ES5中引入了严格模式（strict mode）的概念。严格模式是为javascript定义了一种不同的解析和执行模式，在严格模式下，ES3中的一些不确定的行为将得到处理，而且对某些不安的操作也会抛出错误。要在这整个脚本启用严格模式，可以在顶部添加如下代码：

“use strict”
